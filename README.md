# Speckle Dynamics Analysis Toolkit

This repository contains computational tools for analyzing temporal variations in biological samples using video-based speckle imaging techniques.

## Features
- Frame extraction from videos
- Video metadata analysis (FPS, resolution)
- Activity map generation (Fujii method)
- Frequency-label-based analysis
- PCA for temporal dynamics

## Applications
- Biomedical optics
- Dynamic pattern analysis
- Research reproducibility
